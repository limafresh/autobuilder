# Autobuilder

A simple mod for Luanti that automates the construction of cubes with customizable width, length, and height, and allows you to make the cube empty or filled. Useful for building skyscrapers, filling spaces, etc.

## Launch
Enter the following into the chat:
```
/autobuilder
```
After clicking OK in the GUI, click with the Autobuilder Wand (autobuilder:wand) tool on the block where the structure should begin.
